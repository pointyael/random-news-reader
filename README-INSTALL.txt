Doc installation serveur local nodejs

1) Installer nodejs : https://nodejs.org/en/download/

2) Récupérer le projet sur git : https://github.com/pointyael/random-news-reader

3) Lancer un terminal de commandes à la racine du projet

4) Commande pour lancer le serveur : >run-server.bat

5) Le serveur est accessible sur localhost:3000
